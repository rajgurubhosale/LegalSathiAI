# LegalSaathi 🏛️

**LegalSaathi** makes Indian law accessible to everyone — 
ask any legal question in plain language and get answers 
grounded in the actual law.

Currently building an intelligent legal assistant that lets anyone query 
the Bharatiya Nyaya Sanhita and Bharatiya Nagarik Suraksha Sanhita in plain language.
- 🔜 More acts coming


## 🔭 What I'm Building
- End-to-end RAG pipeline with hybrid retrieval (BM25 + vector search)
- Parent-child chunking strategy that respects legal section boundaries
- FastAPI backend for serving legal queries
